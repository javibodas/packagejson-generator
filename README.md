# packagejson-generator

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/javibodas/packagejson-generator.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Build the project
   ```sh
   npm run build
   ```
3. Run the project
   ```sh
   npm run start
   ```
